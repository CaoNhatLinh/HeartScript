# My Love App ❤️

A private space for us. Built with Next.js, Firebase, and Love.

## Getting Started

1.  **Environment Setup**:
    Copy `.env.example` to `.env.local` and fill in your Firebase credentials.
    ```bash
    cp .env.example .env.local
    ```

2.  **Dependencies**:
    Install the packages:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## Features

-   **Authentication**: Secure login for both of us.
-   **Dashboard**: Central hub for our activities.
-   **Journal**: Shared diary to record our days.
-   **Memories**: Photo and video gallery.
-   **Calendar**: Important dates and "Days in Love" counter.
-   **Projects**: Interactive coding space for 3D scenes (Three.js supported).

## Tech Stack

-   **Framework**: Next.js 14 (App Router)
-   **Styling**: Tailwind CSS
-   **Database**: Firebase Firestore & Storage
-   **Auth**: Firebase Auth
-   **Editor**: Monaco Editor
